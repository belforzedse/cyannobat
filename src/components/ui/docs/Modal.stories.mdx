# Modal

Glassmorphic dialog wrapper powered by Radix Dialog. Customize the tone for different emphasis and size to match content density.

```tsx
<Modal>
  <ModalTrigger asChild>
    <Button>Open modal</Button>
  </ModalTrigger>
  <ModalContent tone="accent" size="lg">
    <ModalHeader>
      <ModalTitle>عنوان</ModalTitle>
      <ModalDescription>توضیحی کوتاه درباره محتوا</ModalDescription>
    </ModalHeader>
    <ModalBody>محتوای اصلی</ModalBody>
    <ModalFooter>عملیات</ModalFooter>
  </ModalContent>
</Modal>
```
